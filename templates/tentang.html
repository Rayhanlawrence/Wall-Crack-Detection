Tentang Proyek

Proyek ini bertujuan untuk mengembangkan sistem deteksi retak pada permukaan beton secara otomatis menggunakan teknologi Deep Learning.

Teknologi yang Digunakan

  • Convolutional Neural Network (CNN)
  • TensorFlow / Keras

Dataset

Dataset yang digunakan adalah Concrete Crack Dataset dari Kaggle.

Cara Kerja Model

  • Training: Melatih model CNN menggunakan dataset retak beton.
  • Testing: Menguji model dengan gambar baru untuk mendeteksi retak.
  • Thresholding: Menentukan ambang batas untuk klasifikasi retak atau tidak.
